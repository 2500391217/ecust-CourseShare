这里列出了`ecust-CourseShare/物理学院` 目录下所有的物理书籍和资料的清单，以便寻找。

如果您在`ecust-CourseShare/物理学院`的任一子目录下添加了书的pdf，务必同步更新这份书单。如果有在github里创建超链接的简单办法可以在issue里建议。

一般来说，教材类书籍放在对应学科的第一级目录下，其他书籍位于 `5 其他书籍和pdf` 目录下。
